개인정보처리방침

“전국인명구조함(베타버전)” 앱(이하 "앱")은 사용자의 개인 정보를 보호하기 위해 최선을 다하고 있습니다. 본 방침은 사용자가 앱을 사용할 때 수집되는 정보, 그 활용 방법 및 보호 방안을 설명합니다.

본 앱은 Google Play 개발자 “안전이 제일이지 말입니다” (gajimala@gmail.com)이 제공합니다.

1. 수집하는 정보

앱은 사용자의 정확한 위치 정보를 수집할 수 있습니다. 해당 정보는 구조 요청 전송 기능 및 지도에서 현재 위치를 표시하기 위해 사용됩니다.

2. 정보의 사용 목적

  • 사용자의 현재 위치 표시
  • 주변 인명구조함 안내 및 구조 요청 시 위치 전송

3. 정보 저장 및 외부 제공

앱은 위치 정보를 서버에 저장하거나 제3자에게 제공하지 않습니다. 모든 정보는 사용자의 단말기 내에서만 일시적으로 처리됩니다.

4. 제3자 서비스 사용

앱은 다음 외부 서비스를 사용합니다:

  • Google Play 서비스
  • 네이버 지도 API
  • 카카오 지도 API

각 서비스는 별도의 개인정보처리방침을 가지고 있으며, 관련 내용은 해당 업체의 정책을 참조하시기 바랍니다.

5. 이용자의 권리

앱은 별도의 회원 가입이나 계정을 요구하지 않으므로, 수집된 개인정보에 대한 삭제 요청은 해당되지 않습니다. 단, 앱 삭제 시 모든 사용 내역은 자동으로 종료됩니다.
